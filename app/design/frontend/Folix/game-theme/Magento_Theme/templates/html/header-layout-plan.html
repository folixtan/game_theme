folix-nav-wrapper display: none 显示 PC 导航

✅ 改造清单

1. XML 修改 (default.xml)

① 恢复货币切换 (currency)
② 移动货币到 folix.top.right
③ 移动 header.links 到 folix.top.right
④ 移动端导航添加语言/货币/登录块
⑤ 确保只有一个 topmenu 块

2. 模板修改

① mobile-nav-header.phtml - 添加语言/货币/登录
② 或创建新模板 mobile-nav-footer.phtml

3. 样式修改 (_header.less)

① 移动端隐藏 header.panel
② 移动端隐藏 folix-nav-wrapper
③ 移动端隐藏 top.search
④ 移动端显示 nav-toggle
⑤ PC端隐藏 mobile-nav-container
⑥ PC端显示 folix-nav-wrapper
⑦ 添加 .switcher.currency 样式

4. JS 交互

① nav-toggle 点击打开侧边栏
② nav-close 点击关闭侧边栏
③ 遮罩层点击关闭侧边栏
④ body 滚动锁定

🔗 Mageplaza 社交登录集成

关键点：
1. 登录链接必须有 href="#social-login-popup"
2. 登录链接会自动添加 .social-login-btn 类
3. 弹窗 #social-login-popup 由 Mageplaza 自动创建
4. 无需修改 Mageplaza 代码，只需确保链接结构正确

登录链接 HTML 结构

<ul class="header links">
    <li class="authorization-link">
        <a href="#social-login-popup" class="social-login-btn">
            登录
        </a>
    </li>
    <li>
        <a href="#social-login-popup" class="social-login-btn">
            注册
        </a>
    </li>
</ul>
